# omacmux-pi

Pi package for the omacmux terminal agent environment.

## Includes

- `extensions/omacmux-pi`: omacmux tool bridge, presets, and safety gates.
- `skills/omacmux`: omacmux operating model.
- `prompts/`: swarm, worktree, review, and handoff prompt templates.

## Install

From this repository:

```bash
pi install ./config/pi
```

For development without installing:

```bash
pi -e ./config/pi
```
